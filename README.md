## Ringkasan
`Form Login ala Hacker` adalah template tampilan login modern dengan estetika "hacker terminal" (neon green, grid animasi, glow). Cocok untuk demo, portofolio, atau digabungkan dengan autentikasi Laravel (Auth::attempt, Breeze, Jetstream).

---

## Fitur
- Tampilan *hacker/terminal* dengan neon green glow.
- Grid animasi berbasis elemen `<span>` (dapat diatur jumlahnya lewat Blade loop).
- Responsif (desktop & mobile).
- Mudah diintegrasikan ke autentikasi Laravel.
- CSS modern (backdrop-filter, animasi, text-shadow).
